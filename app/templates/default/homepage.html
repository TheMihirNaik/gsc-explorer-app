Frequently Asked Questions

Everything you need to know about GSCExplorer.app

Why is GSCExplorer.app completely free?

This is my learning project. I don't have the skills to release a product where I can charge yet, but I want to learn. I just expect this to be useful in your workflow, and you put in more feature requests so I can learn more. Since the GSC API is free, the cost of running this project is just $7 per month.

Is my data secure? Do you store any of my information?

Your data and privacy are our top priorities. This tool calls the GSC API when you hit the "Fetch" button. All data and credentials are stored locally and get removed once you exit the session. There are no database connections in the code. You can verify this by checking our public codebase.

What analytics do you use on the site?

For learning purposes, I've installed Google Analytics 4 (GA4) to understand how the product is being used and to learn about user behavior. This information helps improve your experience.

I've also implemented Microsoft Clarity to gain insights through heatmaps and session recordings, which helps identify areas for improvement and enhance features.

How does GSCExplorer.app overcome GSC's 1000-row limitation?

We use the Google Search Console API efficiently to fetch data beyond the standard 1000-row limitation. Our tool makes multiple API calls and combines the results to provide you with a complete dataset for comprehensive analysis.

Can I contribute to the project?

Absolutely! GSCExplorer.app is an open-source project, and we welcome contributions. Whether you're a developer, designer, or SEO expert, you can help improve the tool. Visit our GitHub repository to get started.
