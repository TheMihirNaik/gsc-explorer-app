Why GSC Explorer is FREE?

This is my learning project. I know I don't have skills to release a product where I can charge yet, but I want to learn. I just expect this to be useful in your worklow, and you put in more feature request so I can learn more. Since, GSC API is free, the cost of running this project is just $7 per month.

Do you store any data?
This tool calls GSC API when you hit "Fetch" button. All the data and credentials are store locally, and gets removed once you exit the session. There are no database connections in the code. Refer to the public codebase to verify.
